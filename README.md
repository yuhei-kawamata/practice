# php-basics-practice

## 概要

COACHTECH 教材 Tutorial 7-1「PHPの基礎 ハンズオン演習」で作成した成果物です。

## 使用技術

- PHP 8.x

## 学んだこと

- 画面に表示させたい数字があるときに、変数から計算できるようにしておくと、後々設定が変更された際に動的に反映させることができる。（消費税１０％を、ベタ打ちで１０％と記載するのではなく、$tax_rate \* 100で表現した。）
- echo文の中に改行タグ（<p>など）を入れないと、一行で表示されて画面が崩れてしまう。

## 動作確認

http://localhost:8000/7-1-6_hands-on/practice/price_calculator.php　にアクセスして、意図通りに表示されていることを確認した。
